HOW TO OPEN THE PROJECT IN ECLIPSE:

Open WordCount.java
Go to Run button on top bar, and click on the arrow next to the green arrow.
Pass the argument as the file you want the program to read and count the number of words
Open TextInfo.java
The program runs.
